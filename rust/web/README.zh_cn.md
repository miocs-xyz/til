## 开发框架
现在有很多用于Web开发的Rust框架，经过比较我最后选择了。
- [actix_web](https://actix.rs/)
actix_web并不视图引擎所以我们要使用askama来生成页面。
- [askama](https://github.com/djc/askama)

那么就让我们通过使用Rust来制作一个简单的CMS程序来看下，如何使用actix_web来构筑一个Web程序。
