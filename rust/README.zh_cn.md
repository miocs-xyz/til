## 安装
Linux用户可以通过下面的命令进行安装
> curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

## 新项目做成
执行下列语句创建一个可执行的Rust程序
> cargo new myproject
> 
> 或者
> 
> cargo new myproject --bin

执行下列语句可以创建一个库程序
> cargo new mylib --lib
